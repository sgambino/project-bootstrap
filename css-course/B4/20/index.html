Bootstrap 4: Font size (px % em rem)

Ex. 1: font-size: 24px Lorem ipsum dolor...

Ex. 2: font-size: 150% Lorem ipsum dolor... note: default font-size is 16px. 16px + 50% = 24px

Ex. 3: font-size: 2em Lorem ipsum dolor... note: 2em is default * 2

Ex. 4: font-size: 2rem Uh-oh! What happened? This looks same as example 3 above. The difference? REM is relative to the browser default font-size. EM is related to the ELEMENT font-size. If you were to change the font-size of the parent element, in this case the <body> tag, the REM setting would not be affected.
